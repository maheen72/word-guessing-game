# Word Guessing Game (Python CLI)

A simple and interactive command-line Word Guessing Game built using Python.
Players select a difficulty level and try to guess a randomly chosen word within a limited number of attempts.

---

## 🚀 Features

* Three difficulty levels: Easy, Medium, Hard
* Random word selection using Python's `random` module
* Limited attempts based on difficulty
* Win/Lose game logic with feedback
* Option to save the secret word to a file
* Play multiple rounds
* Input validation for incorrect choices

---

## 🛠️ Technologies Used

* Python 3
* Random Module
* File Handling
* Loops and Conditional Statements

## 🎮 How to Play

1. Choose a difficulty level:

   * Easy
   * Medium
   * Hard

2. Guess the hidden word.

3. You have limited attempts based on difficulty.

4. If you guess correctly, you win.
   Otherwise, the game ends.

5. You can choose to save the secret word.

6. After the game ends, you can play again or exit.
